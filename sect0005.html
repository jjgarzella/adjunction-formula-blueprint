1.4 Associated Primes

Definition 1.4.1
#

The support of a module is the set of prime ideals \(\mathfrak {p} \subset R\) such that \(M_{\mathfrak {p}} \neq 0\).

Proposition 1.4.2
#

Let \(R\) a ring, and \(M\) an \(R\)-module. Let \(\mathfrak {p}\) be a prime ideal, then the following are equivalent:

  1. \(\mathfrak {p} = \text {Ann}_R (m)\) for some element \(m \in M\).

  2. \(R / \mathfrak {p}\) embeds into \(M\).

Definition 1.4.3
#

Uses

  • Proposition 1.4.2

The set \(\text{Assoc}_R (M)\) is the set of primes satisfying the preivious proposition

Proposition 1.4.4
#

The associated primes of \(M\) are in the support of \(M\).
